Class wrapper around an array of key/value pairs

Here is an example:  
  1.  <?php
    2. 

  2.      $bag = new TChester_StructureBag(); // Creates new bag
    3. 

  3.      $bag->title = "My Title";           // Key/value "title" ==> "My Title"
    4. 

  4.      $title = $bag->title;               // Get "My Title" using "title" key
    5. 

  5.      $name = $bag->name;                 // Error - key does not exist
    6.
