Class Details



[line 240]

Class to extract text contents and metadata out of a document that was  created using the Adobe Portable Document Format.

WARNING: The PDF 1.4 specification allows incremental updates  to the PDF which could result in appearance of multiple body, cross-  reference, and trailer sections. This class has not been tested using  such a PDF file and the output of the public interfaces is unknown. It  is assumed this incremental updating is the exception  rather than the norm.
Here is an example showing how to get the contents and metadata:  
  1.     $object   = new TChester_Pdf2Text("document1.pdf");
    2. 

  2.     $contents = $object->getContent();
    3. 

  3.     $title    = $object->getTitle();
    4. 

  4.     $author   = $object->getAuthor();
    5. 

  5.     $subject  = $object->getSubject();
    6. 

  6.     $keywords = $object->getKeywords();
    7. 

  7.     $creator  = $object->getCreator();
    8. 

  8.     $producer = $object->getProducer();
    9. 

  9.     $created  = $object->getCreationDate();
    10. 

  10.     $modified = $object->getModDate();
    11.
